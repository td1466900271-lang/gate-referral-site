
## 每天更新日报

1. 复制 `daily/2026-06-30/` 为新的日期目录，例如 `daily/2026-07-01/`。
2. 修改新页面的标题、描述、日期和正文。
3. 在 `daily/index.html` 的历史日报列表中加入新链接。
4. 在 `sitemap.xml` 中加入新 URL。
5. 提交并推送到 GitHub，GitHub Pages 会自动部署。

## 重要说明

页面里的注册按钮指向 Gate 官方注册入口，并带有邀请码参数。奖励、返佣和活动资格以 Gate 官方页面当前规则为准。
